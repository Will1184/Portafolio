Spring-Universidad

Proyecto Back-End utilizando API REST. Fue creado para el manejo administrativo de una universidad, mediante el cual se pueden realizar operaciones CRUD que se almacenan en una base de datos, utilizando las entidades de "profesor", "empleado", "alumno", "pabellón", "aula", "materia" y "carrera". Además, incluye documentación Swagger-UI para su demostración en un servidor sin necesidad del uso de un Front-End.

Tecnologias usadas:

Java, Spring, Hibernate, MapStruct, Lombok, Swagger-UI, Postgresql, Base de datos H2
